Game Rules

  • Choose starting coins (default: 500)
  • Random seating order for each question
  • See question and bid as many coins as you want
  • Highest bidder gets to answer first
  • If correct, only half their bid is deducted
  • If wrong, whole bid amount is deducted
  • Question passes to next highest bidders
  • Next bidders don't lose or win coins if correct
  • Non-bidders can win 10% of starting coins if correct
  • Winner: most points, then conversions, then coins
